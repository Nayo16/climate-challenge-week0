# climate-challenge-week0

## Project Overview
This project is part of the 10 Academy Week 0 challenge.  
The goal is to analyze climate data from multiple African countries and extract meaningful insights.

## Setup Instructions

1. Clone the repository:
   git clone https://github.com/Nayo16/climate-challenge-week0.git

2. Navigate into the project folder:
   cd climate-challenge-week0

3. Create a virtual environment:
   python -m venv venv

4. Activate the environment:
   Windows:
   venv\Scripts\activate

5. Install dependencies:
   pip install -r requirements.txt

## Project Structure
- `src/` → source code
- `notebooks/` → analysis notebooks
- `data/` → dataset (not included in repo)
- `.github/workflows/` → CI configuration

## Notes
- Data files are excluded using `.gitignore`
- GitHub Actions is used for basic CI checks
